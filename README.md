# Shortener

A thrown together all in one URL shortener.

## Roadmap
1. Streamline build.
2. Embed the UI for an AIO binary
3. Use a DB
4. Unique generation not SHA1 (Letter, Letter, NUM, Letter, Letter)
5. CSV upload options
6. plain text sharing

## Usage:
1. `npm install`
2. `npm run build`
3. `go run main.go`
